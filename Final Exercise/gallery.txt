[05.11.2014 16:47:38] Roman Smichyk: Задача: розробити галерею для показу довільної кількості зображень.
Формат вхідних даних: JSON масив у вигляді
[
  {
    imageURL: '',
    thumbnailURL: '',
    url: ''
  },
  {
    imageURL: '',
    thumbnailURL: '',
    url: ''
  },
  ...
],
де
  imageURL - шлях до зображення;
  thumbnailURL - шлях до маленького ескіза зображення;
  url - адреса, за якою потрібно перейти при кліку по зображенню.

Основні вимоги:
 - зображення можуть бути різного розміру і співвідношення. Відповідно, розмір галереї визначається автоматично по зображенням, якщо розмір галереї не переданий явно;
 - спосіб анімування зображень: слайд. Але в будь-який момент може стати іншим, тому процес анімування немає мати прямих звязків з основною логікою;
 - для гортання зображень повині бути дві стрілки зліва і справа галереї вирівняні по вертикалі. Вони не повинні виходити за межі галереї;
 - для гортання ескізів теж повинні бути срілки зліва і справа з тими самими умовами;
 - список ескізів показується в залежності від переданих налаштувань (знизу, зверху, зліва, справа);
 - список ескізів повинен автоматично підскролюватися, якщо ескіз поточно показаного зображення виходить за межі області видимості полоси ескізів;
